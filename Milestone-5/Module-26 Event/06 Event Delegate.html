Explore Event Delegate

Event delegate is used to use the parent for using its childs event handler.
using this we dont need to assign event handler to every child nodes
because in event bubble the child hanfler will appear in the parent node by back propagaton.
so its better to use event delegate to only use the parent.
it helps us from assigning multiple event handler in child and optimize ram



Here using event handler in every child and deleting child.
but the parent wont be able to remove the newly added child
even placing the child event handler code at last wont solve the problem
